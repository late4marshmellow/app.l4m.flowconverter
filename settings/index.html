🔑 API Key (required for writing flows)

Homey requires a user-level API key to read and update flows.

How to create one:
1. Open my.homey.app in your browser
2. Go to Settings → API Keys
3. Click New API Key, give it a name (e.g. Flow Converter)
4. Enable exactly these two permissions:
   • Flows — needed to read and update your flows
   • Devices — needed to list devices for the search
5. Click Create, then copy the key (it's only shown once)
6. Paste it below and click Save token

Save an API key first. The converter controls below appear only after a token is present.
